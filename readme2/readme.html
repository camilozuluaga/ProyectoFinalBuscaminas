Instalación: la famosa Instalación en 20 minutos

  1. Abrir el .rar con el programa predeterminado(Winrar, 7Zip,WinZip).
  2. Extraer el archivo con el anterior.
  3. Debe crear una base de datos en la plataforma wampserver como lo explica el siguiente video Tutorial
  4. Doble clic en el ejecutable(.jar) ProyectoFinalBuscaminas.
  5. Empezar a disfrutar de este juego.

Migrar desde otros sistemas

ProyectoFinalBuscaminas ha sido desarrollado en lenguaje de programacion llamado Java, por lo tanto este funcionara en cualquier sistema operativo sin casi ningun problema.

Requisitos del sistema

  • JDK 7.2
  • Base de datos creada con los siguientes requisitos Tutorial

Recomendaciones del sistema

  • Procesador de 1.6 Gz.
  • Memoria RAM 2 Gb.
